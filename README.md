# DataStructures Library
Implementation of different data structures as a C++ library.

## Requirements
- CMake 3.10+
- C++20
- g++ 8+

## Installation
```
git clone git@github.com:James-02/DataStructures.git
cd DataStructures
```

## Build && Run
``` CMake
cd build
cmake ../ . && make && ./main

```
